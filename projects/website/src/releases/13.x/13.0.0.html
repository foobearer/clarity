Highlights

This is the official 13.0.0 release, which is fully compatible with Angular 13.

To update to Clarity v13, first update to Angular 13 following their update guide. Then to update Clarity, run the update command below to have it update necessary dependencies. Finally, review any breaking changes that might impact you and update as necessary. 


ng update @clr/angular@latest

Bug Fixes

  • Fix color contrast of combobox pill border to meet accessibility requirements (PR#6579)
  • Fix ellipsis issue with dropdown toggles in Safari (PR#6569)
  • Fix vertical alignment of icon in dropdown toggle buttons (PR#6565)
  • Improve programmatic opening and closing of alerts (PR#6560)
  • Fix issue with closing first app-alert showing the last alert instead of the next one (PR#6560)
  • Show next alert in app-alerts when closing the first of two (PR#6560)
